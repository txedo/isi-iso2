\section{Introducción}

\subsection{Qué es Symbian}

Symbian es un sistema operativo (conocido como \textit{Symbian OS}) utilizado en dispositivos móviles, como PDAs o teléfonos móviles. Debido a las limitaciones de estos dispositivos móviles, Symbian está diseñado para residir en un espacio muy pequeño, para utilizar dinámicamente los recursos de memoria, para administrar eficientemente la energía y para soportar en tiempo real los protocolos de comunicación y telefonía.
Técnicamente, el sistema operativo Symbian es una colección compacta de código ejecutable y de otros archivos, siendo la mayoría de ellos bibliotecas que se cargan dinámicamente (DLLs) y archivos de configuración, de imágenes y de tipografía. Symbian se almacena, generalmente, en un circuito flash dentro del dispositivo móvil, para que, gracias a esto, se pueda conservar información aunque el sistema no posea carga eléctrica en la batería y para permitir la re-programación, sin necesidad de separar este circuito de los demás circuitos.

Las aplicaciones compatibles con Symbian se desarrollan a partir de lenguajes de programación orientados a objetos, como C++, Java (con sus variantes como PJava, J2ME, etc.), Visual Basic etc. Cada aplicación desarrollada se puede instalar en la memoria flash del dispositivo con Symbian utilizando un PC, una interfaz USB o firewire, dependiendo del modelo de dispositivo y del cable correspondiente. 

Para terminar, Symbian permite actualizarse, tarea que puede llevar a cabo el propio usuario del dispositivo móvil a través de los sitios en Internet de los fabricantes de los dispositivos móviles o bien  al obtener una tarjeta de memoria flash de los distribuidores autorizados, la cuál contiene la nueva versión de Symbian. 

\subsubsection{Breve historia de Symbian}

Symbian OS es una evolución de la plataforma de 32 bits \textit{Psion EPOC} que se utilizó por primera vez en 1997 en el Psion Series 5 PDA. La primera versión de EPOC en el Psion Series 5 se conoce como ''EPOC Release 1'', aunque este número de versión iba incrementando prácticamente cada año, debido a las continuas revisiones que se hacían de esta primera versión.

En 1998, Symbian fue creada como una \textit{spin-off} del \textit{Psion Software Ltd}, co-propiedad de Psion, Nokia, Ericsson y Motorola. El propósito de este \textit{spin-off} fue desarrollar una plataforma de software avanzado para unos nuevos modelos de teléfonos móviles, llamados teléfonos inteligentes, que combinan telefonía y capacidad de cómputo. Esto representó un cambio radical en la plataforma de EPOC, que, un año más tarde, se convirtió oficialmente en la plataforma \textit{Symbian OS}, que es actualmente co-propiedad de diferentes empresas, tales como Motorola, Sony Ericsson y Nokia, entre otros, conocidas como \textbf{Symbian Foundation}. Además, la plataforma Symbian ha sido liberada y en la actualidad es OpenSource.

En cuanto a las versiones de este sistema operativo, el primer lanzamiento de la plataforma Symbian fue la versión Symbian OS 6.0, utilizada en el teléfono \textit{Nokia 9120 Communicator}, en el año 2001, evolucionando hasta la versión actual, la 9.5. De este modo, aunque se hallan añadido nuevas funcionalidades al sistema operativo y se haya modificado su API, la arquitectura del sistema operativo Symbian y los paradigmas de programación utilizados se mantienen cercanos a la plataforma original de EPOC de Psion, por lo que no supone mucho esfuerzo migrar de una versión a otra.
