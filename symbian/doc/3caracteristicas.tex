\section{Características}
Symbian es una plataforma que, desde su liberación, está ganando cada vez más adeptos. No obstante, a día de hoy las características que puede ofrecer este sistema operativo a un terminal no sobrepasan a cualquiera de las que ofrecen sus competidores como iPhoneOS, Android o Windows Mobile:
\begin{milista}
	\item \textbf{Telefonía y mensajería.} De forma adicional a los métodos tradicionales de mensajería y telefonía (mensajes de texto, mensajes multimedia, llamadas de voz y llamadas de vídeo), se ofrecen distintas alternativas como Skype, MSN Messenger, Facebook Mobile, Twitter, Google Talk, etc. También se ofrecen distintos mecanismos para recibir y actualizar servicios de blog, correo electrónico o sindicación Web.
	\item \textbf{Aplicaciones.} Se ofrecen miles de aplicaciones para todo tipo de usos (juegos, productividad, oficina...).
	\item \textbf{Internet.} El sistema incorpora por defecto un navegador Web.
	\item \textbf{Cámara y fotos.} Soporta cámaras de fotos de hasta 12 megapíxeles y ofrece la posibilidad de compartir fotos y videos de forma instantánea mediante Flickr o Facebook, así como por correo electrónico o mensajes multimedia tradicionales.
	\item \textbf{Música y radio.} También es posible convertir el dispositivo en una estación de radio FM o escuchar música por \textit{streaming} o habiéndola descargado previamente.
	\item \textbf{GPS y mapas.} Soporte para GPS y Google Maps.
	\item \textbf{Seguridad.} El sistema incorpora soporta mecanismos de seguridad, tales como reconocimiento de huellas dactilares, antivirus, etc.
	\item \textbf{Open source.} Desde el 4 de febrero del 2010 la plataforma Symbian ha sido liberada. Esto hace posible que cualquier persona pueda desarrollar sus propias aplicaciones o participar en el desarrollo de aplicaciones ya existentes.
	\item \textbf{Vídeo y televisión.} Posibilidad de ver y compartir vídeos de YouTube en alta definición (HD), así como conectar el móvil a la televisión para ver los vídeos en la gran pantalla.
\end{milista}

Pero Symbian se quiere poner a la altura de sus competidores. Para ello, las personas que están detrás de esta plataforma están llevando a cabo numerosas investigaciones para aumentar las posibilidades que puede ofrecer este sistema operativo a sus consumidores:
\begin{milista}
	\item \textbf{Alertas de localización.} El teléfono ejecutará la aplicación deseada cuando el usuario entre o abandone algún área previamente configurada.
	\item \textbf{Conexión instantánea.} El usuario podrá conectar el dispositivo al resto del mundo con un simple \textit{"`golpecito"'}. Por ejemplo para emparejar el móvil con un dispositivo bluetooth o descargar música en el teléfono, únicamente deberá dar un pequeño toque entre ambos dispositivos.
	\item \textbf{Calendarios paralelos.} Uso de calendarios separados para los eventos personales y los eventos profesionales y la posibilidad de sincronizar los calendarios sin la necesidad de un ordenador.
	\item \textbf{Detección de zona horaria.} Cuando el usuario viaje y cambie la zona horaria en la que se encuentra, el teléfono configurará automáticamente la hora gracias a la conexión GPS.
	\item \textbf{Pantalla holográfica.} Escritorio en tres dimensiones para organizar todos los iconos en el espacio.
\end{milista}
